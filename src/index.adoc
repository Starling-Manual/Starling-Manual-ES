// introduction
El objetivo de este manual es proveer de una introducción extensiva a http://www.starling-framework.org[Starling Framework].
Starling es un motor multiplataforma para _ActionScript 3_ que puede ser usado para todo tipo de aplicaciones, enfocándose especialmente en juegos 2D.

ifdef::backend-html5[]
TIP: Puedes descargarte la link:manual.pdf[versión PDF] de este documento.
endif::[]

En esta guía vas a aprender:

* Las tecnologías con las que Starling fue creado y los principios que sigue.
* Cómo elegir una IDE y configurarla para tu primer proyecto.
* Los conceptos básicos como ??display list, eventos y el sistema de animación.
* Técnicas avanzadas, como comenzar a entender el potencial de los programas fragment y vertex.
* Cómo conseguir el mejor rendimiento del framework.
* Qué se requiere para que tu juego corre en dispositivos móviles y tabletas.

Seguí leyendo para sumergirte en estos tópicos!
